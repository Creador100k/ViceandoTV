# StreamGrid — Free Live TV on GitHub Pages

A fully static IPTV browser built on **HTML + CSS + Vanilla JS**, hosted on GitHub Pages. No backend, no Node.js, no PHP.

Data is automatically fetched from [iptv-org/api](https://github.com/iptv-org/api) daily via GitHub Actions and cached as static JSON.

## Features

- 📺 Browse **10,000+ free live TV channels** worldwide
- 🔴 Filter to channels that have **working streams**
- 🌍 Filter by **country** or **category**
- 🔍 **Search** channels in real time
- ▶️ **Play HLS streams** inline with hls.js
- 🔄 **Auto-updates daily** via GitHub Actions
- 📱 Mobile-responsive layout
- ⚡ Fast — all rendering is client-side

## Setup

### 1. Fork or clone this repo

```bash
git clone https://github.com/YOUR_USERNAME/streamgrid
cd streamgrid
```

### 2. Enable GitHub Pages

- Go to **Settings → Pages**
- Source: **Deploy from a branch**
- Branch: `master` (or `main`), folder: `/ (root)`
- Save

### 3. Enable GitHub Actions

- Go to **Settings → Actions → General**
- Allow all actions

### 4. Run the workflow manually (first time)

- Go to **Actions → Update IPTV Cache → Run workflow**

After ~2 minutes the `cache/` folder will be populated and your site will have live data.

### 5. Visit your site

```
https://YOUR_USERNAME.github.io/streamgrid/
```

## How it works

```
GitHub Actions (daily 03:00 UTC)
  │
  ├── curl → iptv-org.github.io/api/channels.json
  ├── curl → iptv-org.github.io/api/streams.json
  ├── curl → iptv-org.github.io/api/guides.json
  ├── curl → iptv-org.github.io/api/countries.json
  └── curl → iptv-org.github.io/api/categories.json
        │
        └── scripts/process_cache.py
              │
              ├── Merges channels + streams + guides
              ├── Filters NSFW & closed channels
              ├── Builds country + category indexes
              └── Writes cache/*.json → git push
                      │
                      └── GitHub Pages serves static files
                              │
                              └── index.html fetches cache/*.json
                                    └── Renders everything client-side
```

## Data sources

- **Channels, streams, guides**: [iptv-org/api](https://github.com/iptv-org/api) (Unlicense)
- **Logos**: [iptv-org CDN](https://iptv-org.github.io/iptv/logo/)
- **Playback**: [hls.js](https://github.com/video-dev/hls.js)

## Notes

- Some streams may be geo-restricted or offline — this is expected with free IPTV data
- The platform falls back to the live iptv-org API if the local cache is missing
- No tracking, no cookies, no ads
